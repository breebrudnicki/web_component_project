HTML Code Polymer Element

I'm the heading I'm the paragraph. Hooray for paragraphs!

The element that shows the code actually consists of two elements: html-code and html-tag

html-code only allows html-tag elements. What you can't see is that there are a few attributes associated with the element. The tag attribute allows you to tell the element what type of tag you want to use. this element says tag='html-tag'. If it said tag='h1' it would look like an h1 tag. there is also a boolean close tag. If you put close then it adds the closing tag.
This html-tag has tag="h1" and close This html-tag has tag="p" and close. The html-tag above has tag="img" and does not have the close tag.
